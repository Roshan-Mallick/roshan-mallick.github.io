#  Roshan Mallick Portfolio Website

This repository contains the source code for my personal portfolio website, hosted using GitHub Pages.

🔗 **Live Website:** https://roshan-mallick.github.io/

---

##  About

This portfolio serves as a central hub for showcasing my technical skills, projects, resume, and professional experience.

It is designed to provide recruiters, hiring managers, and fellow developers with a comprehensive overview of my work and achievements.

---

##  Features

* Responsive and modern user interface
* Professional portfolio layout
* Dedicated resume section
* GitHub activity integration
* Search Engine Optimization (SEO)
* Sitemap and Robots.txt support
* Google Search Console verification
* Fast and lightweight design

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## 💻 Running Locally

### Clone the Repository

```bash
git clone https://github.com/Roshan-Mallick/roshan-mallick.github.io.git
```

### Navigate to the Project Directory

```bash
cd roshan-mallick.github.io
```

### Open the Website

Open `index.html` in your preferred web browser.
